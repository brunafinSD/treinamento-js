Todos os scripts dos arquivos estão no vs code / execução no console

É possível transformar uma frase/texto em array usando .split para definir onde será separado

O split vai quebrar o texto em palavras e armazenar cada palavra na posição do array

para unir novamente as palavras em um texto usa-se o .join

o join vai juntar as posições do array usando o separador que definirmos
